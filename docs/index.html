Core Model

Deposits stay token-specific inside the vault ledger. A TSLA deposit is tracked as TSLA, an AMZN deposit is tracked as AMZN, and withdrawals return the original deposited asset.

TSLA AMZN NFLX PLTR AMD USDG

rINDEX Receipt
